## ⚡️ SUPER FAST SETUP (30 seconds)

### Step 1: Get Your Images
You need:
1. Your headshot photo (save it locally - the blob: URL doesn't work)
2. Afterharvest logo (download from Imgur manually - it's blocked for automated downloads)

**To download the logo:**
- Go to: https://i.imgur.com/izdADAh.png
- Right-click → Save Image As...
- Save as `afterharvest-logo.png`

### Step 2: Run The Magic Script
```bash
python3 add-my-images.py your-photo.jpg afterharvest-logo.png
```

This automatically updates ALL 4 signatures with your real images!

### Step 3: Pick Your Favorite
Open each HTML file in Chrome/Safari to see them with YOUR images:
- `sean-signature-final.html` ⭐️
- `sean-signature-bold.html`
- `sean-signature-kinfolk.html`
- `sean-signature-brutalist.html`

### Step 4: Add to Gmail
1. Open your favorite in browser
2. Select All (Cmd/Ctrl + A)
3. Copy (Cmd/Ctrl + C)
4. Gmail Settings → Signature → Create new
5. Paste (Cmd/Ctrl + V)
6. Save

**DONE!** 🎉

## 🆘 Quick Troubleshooting

**"The script won't run"**
- Make sure you saved your photo and logo locally first
- Use: `python3 add-my-images.py photo.jpg logo.png`

**"Images not showing in browser"**
- Did you run the add-my-images.py script?
- Check that both files exist in the same folder

**"Can't access blob: URL"**
- Blob URLs only work in the browser where you created them
- You need to save/download the image file to your computer first

**"Imgur logo blocked"**
- Go to the URL in your browser
- Right-click the image → Save Image As...
- Then use the saved file

**"Layout looks broken in Gmail"**
- Make sure you copied from the browser (not from text editor)
- Try opening the HTML file in Chrome first, then copy

## 💡 Pro Tips

1. **Compress your images first** - Use https://tinypng.com
   - Target: Photo < 50KB, Logo < 20KB
   - Smaller = faster loading = better experience

2. **Test on mobile** - Send yourself an email and check your phone
   - 60%+ of emails are opened on mobile first

3. **A/B test the designs** - Try different ones for different audiences
   - Creative prospects? Use Kinfolk
   - Enterprise sales? Use Final
   - Making a bold pitch? Use Bold or Brutalist

4. **Keep the email short** - Your signature should complement, not compete
   - 3-4 sentences max in the email body
   - Let your signature do the heavy lifting for brand impression

5. **Be consistent** - Pick ONE and use it for all cold outreach
   - Consistency builds brand recognition

## 🎯 Cold Email Formula That Works

```
Hi [Name],

[One sentence about them/their company that's specific]

[One sentence about what you do + how it relates to them]

[Soft question/CTA]

Best,
[Signature]
```

**Example:**
```
Hi Sarah,

I noticed you've grown [Company]'s Slack community to 5,000+ members—impressive.

We help community builders like you turn that engagement into retention
(our clients reduce churn by ~40% on average).

Worth a quick call next week to share what we're seeing?

Best,
[Your beautiful signature]
```

The signature does 80% of the brand/credibility work, so keep the email brief.
